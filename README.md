# 📰 Fake News Generator

Welcome to the internet's most unreliable news source! This is a chaotic, Python-powered command-line application that instantly generates ridiculous, punchy breaking news headlines on demand. 

It mixes random headline formats with completely fabricated, funny corporate sponsors.

---

## 🚀 Features

* **Dynamic Generation:** Uses randomized selections from structured pools of subjects, actions, and locations.
* **Infinite Loop Mode:** Run it as long as you want; the program cleanly prompts you after every headline to continue or exit safely.

---

## 🛠️ Concepts Implemented

This project is a fun showcase of fundamental Python programming basics:
* **Data Collections:** Managing word banks using Python `lists`.
* **Randomization:** Utilizing `random.choice()` for unpredictable combinations.
* **String Manipulation:** Using clean `f-strings` for formatting and `.strip()` for robust input handling.
* **Control Flow:** Building an interactive `while` loop with custom exit flags.
